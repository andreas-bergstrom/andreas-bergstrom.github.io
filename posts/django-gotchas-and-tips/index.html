Django Tips and Gotchas #1 - reverse() vs reverse_lazy()

Posted at — Apr 17, 2019

Named URL-routes is a handy feature that makes maintenance less of a headache as a codebase grows. In Django there are three aspects to URL-routing:

  • The URL-config itself, commonly urls.py in apps and a main project configuration that imports those.
  • In templates, by using the {% url ‘route-name’ %} template tag to construct URL’s automatically.
  • In views, for example redirects often use named routes.
